About Me

Salutations! I'm Noah Trillizio, a game programmer and designer from Buffalo, New York. I've loved video games for as long as I can remember and have always wanted to be able to give back to the medium that raised me. I've been making games since I was 9 where I would make up card and board games for my Boy Scout troop, before eventually learning to code in high school. Since then have graduated cum laude from Bradley University with a Bachelor's in Computer Science with a concentration in Game Technology and a Minor in Game Design.

Here you can see some of the projects I've had the opportunity of contributing to. I have a strong perseverance for learning new things and taking on challenges, doing the work that others might not want to do but still needs to be done. That's how I ended up learning how to do network programming and PHP. I've always been extremely curious about the technical side of games, which I believe is one of the key factors that has led me to learn the broad skill set I have today.

Sometimes I also like to make websites for the games I make. Like this one, you could say I made it for this little game we call life. If you have any questions please feel free to contact me!
